## 技術要件

### 収集
- RSSフィード自動収集（ゴング・MMAPLANET・イーファイト）
- スクレイピング（団体公式5媒体）
- 自動更新: 30分おき
- イーファイトはMMAカテゴリのみ取得

### 表示形式
- 見出し + 100文字以内サマリー + 元記事リンク
- ソースバッジ（色分け）を各カードに表示
- 著作権対応: 全文転載禁止、元記事へ誘導

### DB
- 記事URL・タイトル・ソース・取得日時を保存
- 重複記事の排除（URL照合）

## X（旧Twitter）自動投稿

### 投稿フォーマット
```
【RIZIN】記事タイトル（30文字以内）

▶ 元記事URL

#RIZIN #MMA #格闘技 #Mニュース
```

### ソース別ハッシュタグ
```
RIZIN      #RIZIN #MMA
UFC        #UFC #MMA
修斗        #修斗 #MMA
DEEP       #DEEP #MMA
パンクラス   #パンクラス #MMA
共通        #格闘技 #Mニュース
```

### 投稿ルール
- 重複投稿禁止（投稿済みURLをDBで管理）
- 最大4件/時間（スパム判定回避）
- 深夜0時〜6時は投稿停止
- 速報記事は即時投稿

### 技術要件
- Twitter API v2
- OAuth 2.0 認証
- 無料枠: 月1,500投稿まで

## 選手戦績ページ

### データソース
- Wikipedia API から戦績テーブルを取得
- 「Wikipedia より見やすい」レベルのUIで表示
- データの更新は Wikipedia に依存

### 表示項目
```
選手名（日本語・英語）
所属団体 / 階級
通算戦績（勝-敗-引分）
KO / 一本 / 判定 の内訳
試合履歴テーブル
  - 日付 / 対戦相手 / 結果 / 方法 / 大会名 / ラウンド
```

### URL設計
```
/fighters              全選手一覧
/fighters/taira-tatsuro  選手個別ページ
```

### ニュースとの連携
- ニュース記事内の選手名にリンクを自動付与
- 記事 → 選手ページ → 関連記事 の回遊導線
